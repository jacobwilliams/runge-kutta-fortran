runge-kutta-fortran: Fixed and variable-step Runge-Kutta solvers in Modern Fortran

Description

This is a work in progress!

Available methods:

Method name Type Order Number of Points Reference
rk4 Fixed-step 4 4 Kutta (1901)
rk7 Fixed-step 7 9 Shanks (1966)
rk8-10 Fixed-step 8 10 Shanks (1965)
rkf78 Variable-step 7 13 Fehlberg(1968)
rkf89 Variable-step 8 17 Fehlberg(1968)
rkv89 Variable-step 8 16 Verner (1978)
rkf108 Variable-step 10 17 Feagin (2006)
rkf1210 Variable-step 12 25 Feagin (2006)
rkf1412 Variable-step 14 35 Feagin (2006)

Compiling

A fmp.toml file is provided for compiling runge-kutta-fortran with the Fortran Package Manager. For example, to build:

fpm build --profile release
